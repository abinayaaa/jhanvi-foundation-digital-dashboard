# Social Impact Beneficiary Dashboard

This project is a digital data platform designed for NGOs to manage shelter beneficiaries, 
track rehabilitation progress, and link them with job opportunities.

## Partner Organizations
- **Jhanvi Foundation** – Implementation Partner
- **Atma Foundation** – Program Enabler
- **SAP Volunteers** – Technical Mentors

## Objectives
- Digitize beneficiary registration and tracking
- Provide job-matching recommendations
- Enable data-driven reporting and analytics
